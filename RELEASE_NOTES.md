# RolledPromptMaker v1.2.0

## 새로운 기능

### 템플릿 Export/Import 기능
- **템플릿 내보내기**: 템플릿을 JSON 파일로 저장하여 다른 기기에서 사용 가능
- **템플릿 가져오기**: JSON 파일로부터 템플릿을 불러올 수 있음
- 집과 회사에서 동일한 템플릿을 사용할 수 있도록 지원

### 앱 아이콘 추가
- 귀여운 앱 아이콘이 추가되었습니다 🎨
- Dock, Finder, 앱 전환기에서 예쁜 아이콘 표시

## 개선사항

### 템플릿 수정 기능 개선
- 템플릿 편집 시 기존 내용이 제대로 로드되도록 수정
- 수정 후 저장 시 변경사항이 즉시 반영됨

### UI/UX 개선
- 중앙 패널(프롬프트 생성기)과 우측 패널(결과 및 기록)의 너비 비율을 7:3으로 조정
- 드래그로 패널 너비를 자유롭게 조절 가능
- 템플릿 목록에 Import 버튼 추가 (↓ 아이콘)
- 템플릿 컨텍스트 메뉴에 "내보내기" 옵션 추가

### DMG 설치 개선
- DMG 파일을 열면 Applications 폴더로 드래그 앤 드롭할 수 있는 UI 제공
- 설치가 더욱 직관적이고 편리해졌습니다

## 설치 방법

### DMG 파일 사용 (권장)
1. `RolledPromptMaker-1.2.0.dmg` 파일 다운로드
2. DMG 파일을 더블클릭하여 마운트
3. 열린 창에서 `RolledPromptMaker.app`을 `Applications` 폴더로 드래그 앤 드롭
4. Applications 폴더에서 앱 실행

### ⚠️ 보안 경고 해결 방법

처음 실행 시 **"Apple은 'RolledPromptMaker.app'에 악성 코드가 없음을 확인할 수 없습니다"** 메시지가 나타날 수 있습니다.

이는 앱이 Apple Developer ID로 서명되지 않았기 때문이며, 악성 코드가 아닙니다. (오픈소스 프로젝트입니다)

**해결 방법 1: 시스템 설정에서 허용 (권장)**
1. 앱 실행 시도 → 경고 창에서 `확인` 클릭
2. `시스템 설정` > `개인 정보 보호 및 보안` > `보안` 섹션
3. "RolledPromptMaker.app이 차단되었습니다" 메시지 옆 `확인 없이 열기` 클릭
4. 경고 창에서 `열기` 클릭

**해결 방법 2: 터미널 사용**
```bash
xattr -cr /Applications/RolledPromptMaker.app
```

**해결 방법 3: Control 키 사용**
1. Applications 폴더에서 앱을 `Control + 클릭`
2. 메뉴에서 `열기` 선택
3. 경고 창에서 `열기` 클릭

### ZIP 파일 사용
1. `RolledPromptMaker-1.2.0.zip` 파일 다운로드
2. 압축 해제
3. `RolledPromptMaker.app`을 Applications 폴더로 이동
4. 앱 실행

## 시스템 요구사항

- macOS 14.0 이상
- Apple Silicon 또는 Intel Mac

## 사용 방법

### 템플릿 내보내기
1. 템플릿 목록에서 내보낼 템플릿 우클릭
2. "내보내기" 선택
3. 저장 위치 선택 후 저장

### 템플릿 가져오기
1. 템플릿 목록 상단의 다운로드 아이콘(↓) 클릭
2. JSON 파일 선택
3. 자동으로 템플릿이 추가됨
